# Programming_Fundamentals_Project_Student_Management_System
"Menu-driven C++ program to manage student data, grades, GPA, and attendance."
C++ Student Management System by Muhammad Furqan 

This repository contains a menu-driven C++ program that allows managing the details of 5 students. The program demonstrates practical use of arrays, loops, conditional statements, and user input/output handling.

Features

View Student Details – Displays all information for a selected student.

Update Marks – Modify marks for a student.

Check Pass/Fail Status – Determines if a student has passed (marks ≥ 40).

View Grade – Assigns grades (A–F) based on marks.

Update Age & Department – Allows updating age and department information.

Scholarship Eligibility – Checks if marks are high enough for a scholarship (≥ 85).

Attendance Requirement – Displays attendance percentage based on department.

Phone Number Management – View or update a student’s phone number.

GPA Calculation – Calculates GPA based on marks using a 4.0 scale.

Exit Option – Clean exit from the program.

Concepts Practiced

Arrays for storing student data

Menu-driven programming using while loop

Conditional statements (if-else) for logic handling

Basic arithmetic and GPA calculation

User input validation for menu choices
